# 电商网站项目计划 (shop-demo)

## 1. 项目概述

**项目名称**: shop-demo 电商网站
**项目类型**: B2C 电商平台
**核心功能**: 商品展示、购物车、订单管理、用户认证等核心电商功能

## 2. 功能需求

### 2.1 用户模块
- 用户注册/登录
- 用户信息管理
- 收货地址管理

### 2.2 商品模块
- 商品分类浏览
- 商品搜索
- 商品详情展示
- 商品评论

### 2.3 购物车模块
- 加入购物车
- 修改数量
- 删除商品
- 购物车结算

### 2.4 订单模块
- 创建订单
- 订单列表查询
- 订单详情查看
- 订单状态跟踪

### 2.5 后台管理模块 (可选简化版)
- 商品管理
- 订单管理

## 3. 技术方案

### 3.1 技术栈
- **后端**: Java + Spring Boot 2.7.x
- **持久层**: MyBatis + MySQL
- **前端**: Vue 3 + Element Plus (简化版，可先用静态页面)
- **构建工具**: Maven

### 3.2 项目结构
```
shop-demo/
├── shop-common/          # 公共模块
├── shop-api/             # API 接口层
├── shop-service/         # 业务逻辑层
├── shop-dao/             # 数据访问层
├── shop-model/           # 实体类
└── shop-web/             # Web 层
```

### 3.3 数据库设计
- 用户表 (user)
- 商品表 (product)
- 分类表 (category)
- 购物车表 (cart)
- 订单表 (order)
- 订单明细表 (order_item)
